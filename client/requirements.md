## Packages
react-markdown | Rendering AI responses with formatting
remark-gfm | GitHub Flavored Markdown support for tables/lists
framer-motion | Smooth animations for UI transitions
date-fns | Formatting timestamps
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind classes efficiently

## Notes
- Authentication is handled by Replit Auth (use-auth.ts)
- Chat streaming uses standard fetch with ReadableStream reader
- Layout uses a sidebar/topbar hybrid for navigation
- Brand fonts: Outfit (Headings) and DM Sans (Body)
